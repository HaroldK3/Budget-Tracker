We have used Github to help keep track of changes as well as our Sprint reports to show when things broke. 

Github helps keep track of any changes anybody makes so if one person breaks the project, 
we can go back to a previous commit and rework the project.

The Sprint Report also helps us keep track of changes in roles as we have kept the process of revolving our scrum master and product owner.
Usually, the product owner in the first week will be the scrum master the next week, 
and the product owner that next week will be the scrum master the week after, and so on and so forth.
